Modular

Most of Silk's functionality is in apps. Add, remove, fork, or create them to build the perfect GUI for you. All apps are built with node.js and web technologies. We even have an experimental app framework that is compatible with apps using Electron!

Graphical alternative to SSH

Access a server remotely using your web browser. Silk provides a text editor, file explorer, task manager, and even a terminal. If you use Digital Ocean to host a node.js app , NDeploy makes it simple.
